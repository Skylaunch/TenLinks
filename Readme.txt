Добрый день, для корректной работы моего приложения необходимо иметь Chrome Browser версии 87.0.4280.88 и выше.

После распаковки базы данных из файла "TenLinksDatabase.bak" и клонирования кода с GitHub необходимо в файлах 
"Startup.cs" и "KeywordContext.cs" изменить в переменных "connectionString" имя моего локального сервера, на имя вашего.

Порядок работы с приложением крайне прост:
1)Запуск приложения
2)Запись фразы для поиска в поле для ввода текста
3)Ожидание
4)Просмотр выведенных ссылок
